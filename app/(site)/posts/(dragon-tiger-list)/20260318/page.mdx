# 3月18日龙虎榜深度解析：算力主线吸金百亿，一文看懂多空资金新动向

3月18日，A股市场延续强势，沪指涨0.32%，深证成指涨1.05%，创业板指涨2.02%。两市成交额达2.06万亿，超3500只个股上涨。算力租赁、CPO、半导体、液冷服务器等科技主线表现活跃。

## 二、各路资金动向深度分析

### 1. 北向资金：积极加仓算力，减仓高位题材

今日沪深股通合计成交2636.57亿元。从个股来看，北向资金呈现明显的结构性特征：

**大幅净买入：**
- 云赛智联：沪股通净买入9532.91万元
- 九安医疗：深股通净买入5712.18万元
- 可川科技：沪股通净买入4487.74万元
- 菲菱科思：深股通净买入1407.33万元

**大幅净卖出：**
- 红宝丽：深股通净卖出1.05亿元
- 朗科科技：深股通净卖出1.01亿元
- 吉林化纤：深股通净卖出8376.86万元

**分析：** 北向资金今日明显加仓算力产业链核心标的。云赛智联作为上海国资背景的算力服务商，受益于阿里云AI算力涨价消息；可川科技则因英伟达发布Feynman芯片带动的光通信题材而受追捧。同时，北向资金对前期涨幅较高的化工股红宝丽、存储芯片股朗科科技等进行获利了结。

### 2. 机构资金：聚焦光模块与硅光，兑现存储芯片

机构资金今日操作分化明显，但主线清晰：

**重点买入方向——光模块/硅光：**
- 可川科技获1机构净买入1.26亿元
- 菲菱科思机构净买入4427.42万元

**重点卖出方向——存储芯片：**
- 朗科科技遭4家机构卖出1.73亿元
- 同有科技虽然整体获机构净买入，但多空博弈激烈

**分析：** 英伟达GTC2026发布Feynman芯片，首次将光通信引入芯片间互联，可降低AI数据中心通信能耗70%以上。这一技术突破直接点燃了光模块和硅光芯片赛道的机构热情。可川科技子公司已具备硅光芯片设计到模块测试的全链条量产能力，成为机构重点抢筹标的。与此同时，存储芯片板块出现明显分歧，朗科科技虽走出3天2板，但机构选择高位兑现，与量化资金形成对手盘。

### 3. 游资动向：东北猛男、欢乐海岸现身算力股

游资今日活跃度有所回升，多路知名游资集中布局算力产业链：

- 东北猛男：净买入云赛智联5403万元
- 欢乐海岸：净买入云赛智联3427万元
- 国泰海通证券武汉紫阳东路：净买入金开新能2.01亿元
- 国泰海通证券成都北一环路：三日净买入酒钢宏兴1.52亿元
- T王：净买入红宝丽2705万元、金牛化工3336万元

**分析：** 游资今日主要围绕两大主线——算力与电力协同。金开新能作为算电协同概念股，受益于AI算力需求扩张带来的电力价值重估预期，获一线游资大手笔买入。云赛智联则同时具备算力租赁、液冷服务器、上海国资三重属性，吸引多路游资齐聚。

### 4. 量化资金：开源证券西安太华路表现活跃

量化资金今日参与度较高，开源证券西安太华路成为焦点：

- 朗科科技：净买入约7000万元（与4家机构形成对手盘）
- 金开新能：净买入1.06亿元
- 顺灏股份：与另一量化席位合计买入1.49亿元

**分析：** 量化资金在朗科科技上与机构形成鲜明对立——机构卖出、量化买入，显示出不同资金属性对存储芯片板块的分歧判断。在金开新能上，量化与一线游资形成合力，共同推升股价。

## 三、典型案例：多路资金博弈的微观视角

### 1. 朗科科技：机构与量化的对手盘

朗科科技今日20cm涨停，走出3天2板，但龙虎榜显示：

- 买入方：量化资金（开源证券西安太华路）买入约7000万元
- 卖出方：4家机构合计卖出1.73亿元 + 深股通卖出1.01亿元

**解读：** 存储芯片概念反复活跃，但机构选择在股价高位兑现利润，量化资金则接盘博弈短线情绪。这种机构与量化的对手盘格局，反映出板块内部的分歧加剧。

### 2. 云赛智联：游资+北向合力做多

云赛智联今日涨停，龙虎榜显示：

- 沪股通净买入9532.91万元
- 东北猛男净买入5403万元
- 欢乐海岸净买入3427万元
- 机构净卖出2880.25万元

**解读：** 云赛智联的上涨由北向资金和游资共同驱动，机构则小幅兑现。公司参股的上海智能算力科技公司将形成万卡规模的智能算力供应，直接受益于阿里云AI算力涨价消息，成为资金追捧的核心逻辑。

### 3. 可川科技：机构主导的硅光龙头

可川科技今日涨停，走出3天2板：

- 1机构净买入1.26亿元
- 沪股通净买入4487.74万元

**解读：** 可川科技是今日机构净买入金额最高的个股。公司在硅光芯片和高速光模块领域的布局，完美契合英伟达Feynman芯片带来的光通信技术变革方向，成为机构资金重点配置的标的。

## 四、总结与展望

**今日资金动向的核心特征：**

1. **主线高度聚焦：** 各路资金围绕AI算力产业链展开布局，尤其是英伟达Feynman芯片带动的光通信/硅光方向，以及阿里云涨价催化的算力租赁方向。
2. **分歧开始显现：** 存储芯片板块出现机构与量化的明显对立，显示经过连续上涨后，部分资金选择兑现，短线博弈加剧。
3. **北向资金态度积极：** 沪股通、深股通在算力核心标的上呈现净买入，对行情形成支撑。
4. **游资与量化合力：** 在算电协同等新题材上，游资与量化资金形成合力，推动金开新能等个股放量上涨。

**后续关注方向：**
- 英伟达Feynman芯片带来的光通信技术变革能否持续发酵
- 阿里云涨价对算力租赁板块的催化效应
- 存储芯片板块在机构与量化博弈后的方向选择
